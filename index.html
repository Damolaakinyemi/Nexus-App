📧 Email Verification (EmailJS)

Connect EmailJS to send real 6-digit verification codes by email when users sign up. Without it, codes appear in an in-app modal (works fine for testing).

Quick Setup Guide
  1. Go to emailjs.com → create a free account
  2. Add an Email Service (Gmail, Outlook, etc.) → copy the Service ID
  3. Create an Email Template with these variables:
    {{to_email}} {{otp_code}} {{user_name}}
  4. Copy your Public Key from Account → API Keys
